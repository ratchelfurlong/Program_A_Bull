Problem 29

Problem Difficulty: Hard

Description:

Madhan is a PhD student trying to figure out his H-index. An H-index is defined as an index that attempts to measure both the productivity and impact of the published work of a scientist or scholar. Given the number of papers written by Madhan and the number of other papers which cite his work, the H-index will be the maximun n such that: Madhan has n papers with at least n citations. Madhan is impatient and wants his H-index as fast as possible, so solve this problem in linear time.

You will be given a sequence of integers representing the number of citations for each of Madhan's papers. Output the H-index (as an integer). You will only receive full points if you can output the correct H-index in linear time. If the program takes more than linear time but still gives the correct H-index, you will receive half points. The end of inputs and outputs our denoted by -1.

Sample Input:

5 3 2 3 4 -1

Sample Output:

3
