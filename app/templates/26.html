Problem 26

Problem Difficulty: Hard

Description:

Write a program that will compute the derivative of the given function. An exmple of the function is as follows: "dy (5x ^ 2) + (7y ^ 2) - 8"; this string states that you should take the derivative of the function with respect to y (partial derivative). Every exponent clause will bbe covered by a set of parenthesis (hint on how to parse the string)

Hints:
Multiplication rule is: (fg)` = f `g + fg` (` denotes the derivative of whatever is being primed)
Chain Rule: if (f(x) == h(g(x))) then f`(x) = h`(g(x)) * g`(x)
Partial Derivative: Take the derivative on the variable that matches the 'd/d*' andtreat everything else as a constant.
d/dx sin(x) = cos(x)
d/dx cos(x) = -sin(x)
d/dx e^x = e^x
d/dx ln(x) = 1/x
d/dx a^x = (a^x) * ln(a)
Depending on the function, you might need to implement multiplication rule, the chain rule or implemet partial dericatives to obtain the answer.
The only operators that are going to be used is '+', '-', '*' (so there is no need to implement the division rule)

Sample Input:

dx (6x ^ 2) + (7x ^ 3)
dx (sin(2x))

Sample Output:

(12x) + (21x ^ 2)
2 * (sin(2x))
