Problem Difficulty: Easy

Description:

Let's define the Hamming distance between two equal length strings be the number of differing bits between the strings.
Whitespaces are counted as a character in this problem (e.g. 'Hello' and 'I eat' both have 5 characters in it).

Write a function that computes the Hamming Distance between two equal strings. If the strings aren't equal, output '-1'.

Sample Input:

this is the first string
Hello! I am B@B oth3r!!!

Sample Output:

71
