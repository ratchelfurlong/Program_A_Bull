Problem 22

Problem Difficutly: Hard

Description:

We want to host a game of tug of war. In order to make it fair, we would like to make the teams as fair as possible.
Your job is the make the teams as even as possible. Each line of input will signify a skill level of a person waiting to be selected for a team and when making the teams, the amount of people on a team must only differ by 1.

Input format: The first line of each test case signifies how many participants for the round. Each line afterwards till the end of the test input will be the strength number for that participant. Test inputs will be seperated by a newline character. Each test case will terminate with a -1, but be sure to account for multiple test cases.
Output format: Print the total skill level of each team separated by a space, with the team with the lower skill level printed first (separated by a space).

Input:

5
100
200
20
80
160
-1

Sample Output:

280 280
